# Portfolio v2 — Atul Biju

> 🚧 **Work in progress** — actively being improved and expanded.

Personal portfolio built with React, Vite, and Framer Motion. Designed to be fast, minimal, and interactive — with a focus on animations and a clean dark aesthetic.

Live at **[icarus13.in](https://icarus13.in)**

---

## Stack

- **React 18** + **Vite 5**
- **Framer Motion 11** — scroll reveals, blur transitions, animated counters
- **Lenis** — smooth scroll
- **Lucide React** — icons
- **CSS Variables** — dark/light theming

## Features

- Custom magnetic cursor
- Word-by-word hero text reveal
- Animated stat counters
- Mouse-tracking spotlight on project cards
- Dual-row marquee strip
- Film grain overlay
- Mobile-responsive layout with desktop recommendation popup

## Run locally

```bash
npm install
npm run dev
```
